# 05 Session Builder

Use `00-shared-brief.md` as the base direction.

Create the mobile session builder screen for `MyoRep Timer`.

## Goal
- Turn the current horizontal, drag/drop-first desktop builder into a touch-first mobile composition flow.
- Make add, edit, reorder, and start actions obvious.

## Must Show
- Session summary or overview.
- A mobile-friendly list or stack of session nodes.
- Clear affordances for adding workout nodes and rest nodes.
- A touch-friendly reordering model.
- A strong start-session action.

## Intent
- This is the highest-risk mobile surface and should feel purpose-built for mobile.
- Prefer a vertical composition model over a desktop-style horizontal canvas.

## Notes
- Reordering should not depend on desktop drag-and-drop as the primary interaction.
- This screen should make it obvious how a user edits a node and how the sequence flows.
