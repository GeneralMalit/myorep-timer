# 04 Library

Use `00-shared-brief.md` as the base direction.

Create the mobile library flow for saved workouts and saved sessions.

## Goal
- Replace the current desktop-style sidebar control panel with a clean mobile library surface.
- Make load, duplicate, save-as, rename, delete, import, and export actions understandable without clutter.

## Must Show
- A unified mobile library surface for workouts and sessions.
- A simple way to switch between saved sessions and saved workouts.
- Clear actions for load and manage behavior.
- A mobile-safe import/export entry point.

## Intent
- This should feel like a focused library, not a settings sidebar.
- Use progressive disclosure so management actions do not overwhelm the screen.

## Notes
- Workouts and sessions can live in one screen with segmented switching or another similarly compact mobile pattern.
- Preserve the sense that the timer is still the main product, and the library is support tooling.
