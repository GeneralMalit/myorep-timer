# 03 Timer Runtime

Use `00-shared-brief.md` as the base direction.

Create the mobile timer/runtime screen for `MyoRep Timer`.

## Goal
- Make the countdown and current phase the unquestioned visual priority.
- Reduce surrounding metadata and chrome so the timer feels immersive and operational.

## Must Show
- Large timer/ring display.
- Current phase or block label.
- Supporting status for set/rep or activation/myo context.
- Clear pause/resume and terminate controls.

## Intent
- The timer should feel focused, athletic, and calm under pressure.
- Supporting chips or badges should help, not crowd the screen.

## Notes
- Keep the timer visually dominant above all else.
- This screen should feel viable on small mobile screens without looking cramped.
