# 07 Settings

Use `00-shared-brief.md` as the base direction.

Create the mobile settings flow for `MyoRep Timer`.

## Goal
- Make settings feel supportive and easy to scan on mobile.
- Reduce the current desktop-style density while keeping important controls accessible.

## Must Show
- Audio-related settings.
- Display-related settings.
- One or two numeric or configurable workout behavior settings.
- A clear overall grouping strategy for settings categories.

## Intent
- This should feel like a mobile sheet or screen with strong grouping and high tap comfort.
- Settings should not visually overpower the main timer workflow.

## Notes
- Preserve the premium dark look, but simplify the visual density.
- One-handed use should feel plausible.
