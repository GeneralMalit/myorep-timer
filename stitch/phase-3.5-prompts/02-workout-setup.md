# 02 Workout Setup

Use `00-shared-brief.md` as the base direction.

Create the mobile workout setup screen for `MyoRep Timer`.

## Goal
- Make setup fast, touch-friendly, and easy to scan.
- Keep the user focused on configuring a workout and starting it quickly.

## Must Show
- Fields for total cycles, activation reps, activation pace, rest interval, myo reps, and myo pace.
- A lightweight mode switch between workout setup and session builder.
- A quick voice-guidance control without forcing entry into full settings.
- One dominant primary CTA for starting the protocol.

## Intent
- This should feel like a fast pre-lift control panel, not a dense admin form.
- Make spacing, hierarchy, and touch targets clearly mobile-first.

## Notes
- The workout start action should be visually dominant.
- Keep the screen clean enough that the user can configure it one-handed on a phone.
